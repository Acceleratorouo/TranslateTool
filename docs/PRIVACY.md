# 隐私政策（PRIVACY）

> 最后更新：2026-06-17 · 适用版本：v1.x

TranslateTool（以下简称"本应用"）是一款**完全本地化**的桌面翻译工具。本隐私政策说明本应用如何收集、使用、存储和保护您的数据。

## 1. 我们收集哪些数据

| 数据类别 | 具体内容 | 收集时机 | 存储位置 | 用途 |
|---------|---------|---------|---------|------|
| 翻译文本 | 您复制或粘贴的原文 | 剪贴板监听启动后 | **仅内存**，不落盘 | 翻译请求的输入 |
| 翻译结果 | 翻译引擎返回的译文 | 翻译完成后 | 仅内存 | 显示在悬浮窗 |
| 翻译历史 | (原文, 译文, 时间, 引擎) | 用户点击"保存到历史"或配置开启 | `%LOCALAPPDATA%\TranslateTool\History\` | 历史窗口查询 |
| 翻译缓存 | (原文, 译文, 语言, 引擎) | 每次翻译成功后 | `%LOCALAPPDATA%\TranslateTool\Cache\` | 7 天内避免重复请求 |
| 应用设置 | API Key、主题、语言、快捷键 | 您修改设置时 | `%APPDATA%\TranslateTool\settings.json` | 跨会话保留配置 |
| OCR 语言包 | Tesseract `.traineddata` 文件 | 首次使用 OCR 时 | `%LOCALAPPDATA%\TranslateTool\tessdata\` | 本地 OCR 识别 |

> **我们不收集**任何用户身份信息、设备指纹、IP 地址（除 API 服务端临时记录）、位置、通讯录等内容。

## 2. 数据如何传输

本应用是**翻译请求的代理**，不代理任何其他流量。翻译请求的路径如下：

```
[用户剪贴板 / 框选] → 本应用 → 第三方翻译 API (Baidu/Google/Microsoft/DeepL)
                                                    ↓
                                          仅服务方日志记录
```

- **传输层**：所有请求使用 HTTPS（TLS 1.2+），中间无法被窃听。
- **存储**：服务方（百度/Google/微软/DeepL）的日志策略由其各自隐私政策约束，**本应用不向任何自建服务器上传数据**。

## 3. 您的权利（GDPR / CCPA）

欧盟 GDPR 和加州 CCPA 赋予您以下数据主体权利（Data Subject Rights）。本应用在**应用内**提供完整实现：

| 权利 | GDPR 条款 | 应用内操作 |
|------|-----------|-----------|
| 知情权 | Art. 15 | 设置 → 隐私与数据 → 📊 查看数据摘要 |
| 访问权 | Art. 15 | 设置 → 隐私与数据 → 📤 导出我的数据（ZIP/JSON） |
| 被遗忘权 | Art. 17 | 设置 → 隐私与数据 → 🗑️ 删除所有本地数据 |
| 数据可携权 | Art. 20 | 设置 → 隐私与数据 → 📤 导出我的数据（标准 JSON 格式） |
| 更正权 | Art. 16 | 直接在悬浮窗/设置中修改后保存 |
| 限制处理权 | Art. 18 | 关闭剪贴板监听/框选翻译功能即可 |

**数据导出格式**：包含 `metadata.json`（应用版本、导出时间、所有设置）、`translation_cache.json`、`translation_history.json`。

## 4. 数据保留期限

| 数据 | 默认保留期 | 用户可清除 |
|------|-----------|-----------|
| 翻译缓存 | 7 天（自动过期） | 导出/删除时立即清空 |
| 翻译历史 | 最多 100 条 | 随时清空或一键删除 |
| 应用设置 | 永久（直到用户主动卸载） | 一键删除 |
| OCR 语言包 | 永久（直到用户主动删除） | 一键删除（可保留） |

**应用卸载时**：包管理器（winget/chocolatey/MSI）通常会询问是否删除用户数据。若选择不删除，下次重装时可继续使用。

## 5. 第三方服务

本应用集成的翻译引擎：

- **百度翻译开放平台** — <https://api.fanyi.baidu.com/>
- **Google Translate** — <https://cloud.google.com/translate/docs>
- **Microsoft Translator** — <https://learn.microsoft.com/azure/ai-services/translator/>
- **DeepL API** — <https://www.deepl.com/pro-api>

各服务方对您的翻译请求内容的处理，请参阅其各自的隐私政策。本应用**不向任何自建服务**上传您的数据。

## 6. 未成年人

本应用不针对 13 岁以下儿童，亦不主动收集未成年人信息。

## 7. 政策变更

如有重大变更，会在 GitHub Releases 和首次启动弹窗中告知。继续使用即视为接受新政策。
