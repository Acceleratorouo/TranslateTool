# Changelog

## 1.1.1 - 2026-06-18

### Fixes
- 修复首次运行配置完成后无法自动进入主程序的问题
- 修复设置窗口快捷键清除按钮文字被挤出可视区域的问题
- 修复 AI 翻译供应商模板保存相关的问题
- 修复设置窗口语言选择和区域翻译快捷键的交互问题

### UI
- 移除主悬浮窗左下角的 "Ctrl+Shift+T 唤出" 提示文字
- 优化设置窗口布局：新增源/目标语言选择、区域翻译自定义快捷键
- 更新应用图标和显示图片资源

### AI Translation
- 为 AI 翻译设置页添加模型选择器下拉框
- 添加本地 Gemma 4 E4B 部署进度条和引导
- 完善 AI 翻译功能的本地化支持
- 在设置窗口中嵌入 LLM 供应商配置页面

### Performance
- 移除 AI 设置页嵌套 ScrollViewer，缓解滚动卡顿

## 1.1.0 - 2026-01-15

### Features
- 新增 AI 翻译引擎支持，集成 LLM 供应商管理
- 支持 OpenAI 兼容客户端、Anthropic Messages API
- 支持百度、Google、微软、DeepL 多引擎翻译
- 支持划词翻译、框选翻译、文件翻译
- 支持深色模式和界面语言切换
